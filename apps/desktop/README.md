# apps/desktop (Tauri placeholder)

Este diretório está preparado para receber o app desktop com Tauri quando o módulo offline for priorizado.

## Bootstrap sugerido

```bash
pnpm dlx create-tauri-app@latest apps/desktop
```

Sugestão de stack: frontend com Next.js exportado/static ou Vite + React, com sincronização local para modo offline.
